# 🎓 Dokumentacja ATP System

::alert{type="info"}
**ATP (Advanced Training Platform)** to nowoczesny system zarządzania treningami sportowymi w wersji v0.1.0, który obecnie oferuje zaawansowane zarządzanie użytkownikami i bezpieczeństwo jako fundament dla przyszłych funkcji treningowych.
::

## 🎯 Główne funkcjonalności (v0.1.0)

::list{type="success"}
- **Zarządzanie użytkownikami** - Rejestracja, uwierzytelnianie i profile użytkowników
- **System ról i uprawnień** - Kontrola dostępu z rozróżnieniem na role  
- **Bezpieczne uwierzytelnianie** - System JWT z zarządzaniem wielourządzeniowymi sesjami
- **Responsywny interfejs** - Optymalizacja dla wszystkich urządzeń
- **Zaawansowane bezpieczeństwo** - Automatyczne wylogowanie i kontrola sesji
- **Dokumentacja** - Kompletne przewodniki i instrukcje
::
